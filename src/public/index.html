Todo List API


Endpoint Description Request Body Response Type
GET /todos Return all todos. Null application/json
GET /todos/:id Return a specific todo with the corresponding id Null application/json
GET /todos/overdue Return a list of overdue todos or an empty list if there are no overdue todos. Todos can be filtered based on their due date attribute Null application/json
GET /todos/completed Return a list of completed todos or an empty list, if no todos have been completed. Todos can be filtered based on their due date attribute. Null application/json
POST /todos Add a new todo to the todo list 
{"name":"...", "due":"..."}
 application/json
PATCH/todos/:id Edit the name and/or due date attributes of a todo. 
{"name":"...", "due":"..."}
 application/json
POST /todos/:id/complete Update todo, set attribute complete to true Null application/json
POST /todos/:id/undo Update todo, set attribute complete to false Null application/json
DELETE /todos/:id Deletes a todo by id Null application/json
